## Descrição da Atividade
Esta atividade consiste no upgrade do projeto desenvolvido na Avaliação Diagnóstica.  
Foi criado um formulário de contato em uma nova página chamada **contato.html**, permitindo que o cliente entre em contato após visualizar o currículo.

O formulário possui campos obrigatórios e adicionais, além de botões para limpar os dados e o envio da mensagem.

Ao clicar no botão **Enviar**, o usuário é redirecionado para uma página de confirmação chamada **sucesso.html**, que informa que a mensagem foi enviada com sucesso.

## Tecnologias Utilizadas
